# Django AutoDeploy Project

Автоматическое развертывание Django приложения через Semaphore CI/CD.

## Структура
- `ansible/` - Ansible плейбуки
- `docker/` - Docker конфигурации
- `src/` - Django приложение
- `.semaphore/` - Конфигурация Semaphore

## Быстрый старт
1. Настройте Semaphore проект
2. Запустите template "Deploy Django Locally"
